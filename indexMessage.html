DevBox - Message Plugin

Show pretty messages and easy for developers!

DOWNLOAD

Usage:

								
    <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
	<link href="devBox.message.css" type="text/css" rel="stylesheet">
	<script src="jquery.devBox.message.js"></script>

Example 1: Normal way

Show Me
I'm the div "handle1"
							
	$("#handle1").showMessage({
		//condition: (true: the message appears) | (false: the message hide) | (default true)
		message: "What the fucking are you doing man?",
		type: "error" // success|warning|info
		//delay: time for the message disappear (in milisseconds)
	});
	//Short way: $("#handle1").errorMessage("What the fucking are you doing man?");

Example 2: Short way

Show Me
I'm the div "handle2"
						    
	//                           condition                         message
    $("#handle2").successMessage(new Date().getSeconds() % 2 == 0, "I appear only if the second is par, hahaha!");

Example 3: Short way with delay

Show Me
I'm the div "handle3"
						    
	//Can be: $("#handle3").warningMessage(condition (true|false), message, delay); too
    $("#handle3").warningMessage("I'll find you, and I'll kill you!", 2000);
